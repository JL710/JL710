list

Es gibt verschiedene list Tags. <ul> ist eine unsortierte Liste, diese hat normaler weise einfache Punkte als Symbol. Unterpunkte werden mit dem <li> Tag in einer <ul> definiert.

Mit list-style-type kann man das Symbol festlegen(z.B. none).


aside

Heftet den "Content" an einen bestimmten Ort an (z.B. an die Seite). Dies eignet sich besonders gut für Inhaltsverzeichnisse. Andere Objekte formen sich automatisch um den angehefteten Bereich an.

<aside style="float: left;">Content</aside>

Linux / rpi


Firewall

Die einfachste Firewall ist ufw.

sudo apt install ufw
ufw kann mit sudo ufw enable aktiviert und mit sudo ufw disable deaktiviert werden.
Es ist wichtig zu wissen, dass man teils manuel Ports freigeben muss.
sudo ufw allow [port]
sudo ufw deny [port]
Für den SSH port gibt es den Shortcut sudo ufw allow ssh.

Der Status der Firewall kann mit sudo ufw status.

Es ist auch möglich Regeln für spezifische IP-Adressen fest zu legen.
sudo ufw deny from 192.168.2.1 port 30
