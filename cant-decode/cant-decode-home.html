Can't decode
Can't decode is an encryption software. You can decrypt and encrypt text on high level encryption. Another feature is a password manager. The future of password management.


Features
  • dencoding and encoding for text with code/codefile
  • passwordmanager (different devices possible)


Download:
Download for Windows.


Crossplatform:
Crossplatform is possible!
If you want to use the passwordmanager on multible devices, you need to copy the masterkey. There is a passwortmanager_mastercode file in the maindirectory. You need to copy that file and paste is in the same position on the other device. When you are done, everything should work.
